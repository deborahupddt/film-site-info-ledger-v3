# film-site-info-ledger-v3
该仓库主要用于建立影视类网站的信息台账，对公开网络中存在的电影与电视剧站点进行基础资料整理。记录内容包括站点名称、访问链接、站点类型及简要定位说明，便于集中管理与长期留存。本项目不提供影视内容服务，也不参与资源传播或技术解析，仅作为信息整理与资料记录存在。所有资料整理自公开渠道，不保证实时准确性，与所记录站点不存在任何隶属或合作关系。
